# B20 Exchange: Institutional Web3 Ecosystem

Welcome to the B20 Trading Platform—a professional-grade Decentralized Exchange and Token Incubation Terminal on the **Binance Smart Chain (BSC_56)**.

## 1. Core Ecosystem Modules
*   **Token Creation Hub:** Launch Standard, Bonding Curve, and Fair Launch tokens via the **B20 Factory** (`0xc4F...aCe2`).
*   **Instant Exchange:** Swap any asset on BSC with real-time PancakeSwap V2 Router integration.
*   **Perpetual Futures:** High-leverage derivative trading (up to 100x) with real BNB PnL settlement from the Treasury.
*   **Audit-Ready Analytics:** Comprehensive on-chain ledger, Performance Metrics, and a visual **PnL Heatmap Calendar**.
*   **Market Intelligence:** Integrated AI Agent panel powered by Google Gemini Pro & DeepSeek-V3.

## 2. Technical Stack & Infrastructure
*   **Frontend:** Next.js 14, Tailwind CSS, Framer Motion, Ethers.js v6.
*   **Backend:** Node.js, Express, SQLite3 (Persistent Audit Logging).
*   **Automation:** 24h Treasury monitoring, Automated BSCScan verification, and TrustWallet asset submission service.
*   **Treasury Wallet:** `0x6451EE4DEf4a8b8FbC2c64301A79e267De378935`.

## 3. Deployment & Setup
### Backend
1.  Navigate to `/backend`
2.  `npm install`
3.  `npm run dev`

### Frontend
1.  Navigate to `/frontend`
2.  `npm install`
3.  `npm run dev`
